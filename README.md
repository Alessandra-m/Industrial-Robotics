### 3. Определение расстояния до ArUco маркера
Задание: Выбрать ArUco маркер, расположить его на базе робота и реализовать определение расстояния от камеры до маркера

- Код и результаты: [ArUco/](./ArUco/)
- Алгоритм: Определение пространственного положения маркеров относительно системы координат робота с использованием калибровочного маркера известного положения через каскадные преобразования систем координат:

 1. {M1} → {cam}: T_cam_M1 = solvePnP(маркер_3)
 2. {base} → {M1}: T_base_M1 (известно экспериментально)
 3. {base} → {cam}: T_base_cam = T_base_M1 × inv(T_cam_M1)
 4. {M2} → {cam}: T_cam_M2 = solvePnP(маркер_15)
 5. {base} → {M2}: T_base_M2 = T_base_cam × T_cam_M2

- Параметры:
Калибровочная матрица K: [2585.29, 0, 716.34; 0, 2582.12, 620.85; 0, 0, 1]
Коэффициенты дисторсии: [-0.0632, 0.4798, 0.0076, 0.0035, 0.0000]
Размеры маркеров: M1 = 100.0 мм, M2 = 28.0 мм
Известные координаты M1: (-162.0, -274.0, 2.5) мм

- Результат:
X: -94.67 мм
Y: -455.36 мм
Z с захватом: 70.99 мм (маркер: 52.99 мм)
YAW: 70.11°
